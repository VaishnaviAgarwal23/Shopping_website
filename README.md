This repository contains the source code for a simple, static E-commerce website named "ShopZone." It is built using HTML, CSS, and basic JavaScript.

üìÇ File Structure
The project is organized into the following files:

File Name	Type	Description
website.html	HTML	The main landing page, including a header, navigation, banner, and image collage.
home.html	HTML	The main content page for the Home section, featuring a banner and product categories.
product.html	HTML	Displays the list of "Featured Products" available for purchase.
shoppingcart.html	HTML	A placeholder page showing items currently in the shopping cart and the total.
checkout.html	HTML	The checkout form page for users to enter shipping and payment information.
login.html	HTML	The user login page with an embedded form.
signup.html	HTML	The user account creation (sign up) page with an embedded form.
style.css	CSS	Contains all the styling for the website (layout, colors, fonts, responsive design).
script.js	JavaScript	Contains the validation logic for the login form.
‚ú® Features
The ShopZone website includes the following key features:

Responsive Navigation Bar: Links to all major sections of the site (Home, Products, Shopping Cart, Checkout, Login).

Product Listings: The product.html page displays a list of featured products with images (emojis), names, prices, descriptions, and "Add to Cart" links.

Shopping Cart View: The shoppingcart.html page shows a mock-up of items added and calculates a total price (static for this version).

Checkout Form: The checkout.html page provides fields for Full Name, Email Address, Phone Number, Shipping Address, and Payment Method.

User Authentication: Dedicated pages for Login and Sign Up.

Login Validation: Uses script.js to ensure both fields are filled and the password is at least 5 characters long.

Sign Up Validation: Uses inline JavaScript to ensure all fields are filled and that the Password and Confirm Password fields match.

Consistent Design: Utilizes style.css for a uniform, colorful, and modern look across all pages.

üìù Notes and Next Steps
This is a static representation of an e-commerce site. For a fully functional e-commerce platform, the following would need to be implemented:

Backend Integration: Connect the forms and cart functionality to a server-side language and database.

Dynamic Cart: Implement JavaScript logic to dynamically add, remove, and update products in the shopping cart.

Payment Processing: Integrate with a secure payment gateway for actual transactions.

User Persistence: Implement proper user session management and password hashing for secure authentication.
